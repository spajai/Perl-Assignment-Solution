## question 1

Write one perl module that will transform an Arabic numeral (https://en.wikipedia.org/wiki/Arabic_numerals) into a Roman numeral (https://en.wikipedia.org/wiki/Roman_numerals).

Input: 1776
Output: MDCCLXXVI

## question 2

Write a module that will read a sequence of unordered numbers (between 1 and 10_000) from a file. Given the assumption that one number is missing, the program should determine which number is missing from the sequence.
The numbers can be separated by these characters: spaces (\s+), comma (,), colon (:), semi-colon (;), LF (\n), CR + LF (\r\n)
